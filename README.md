## 📦 Installation & Setup  

1. Clone this repository:  
   ```sh
   git clone https://github.com/your-username/task-manager.git
   cd task-manager
   ```
2. Install dependencies:  
   ```sh
   npm install
   ```
3. Run the application:  
   ```sh
   npm run dev
   ```
